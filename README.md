# Spotify (MoonLoader / SA-MP Mobile)

Музыкальный плеер в стиле Spotify для **MoonLoader** (Android). Треки берутся
из Google Sheets (общий плейлист), из ваших личных плейлистов и из локальных
mp3-файлов на телефоне.

Команда в игре: **`/spotify`**

## Установка

1. Установите MoonLoader для Android и нужные библиотеки:
   `mimgui`, `fAwesome6_solid`, `inicfg`, `requests`, `lfs`, а также
   нативную `libbass.so` (BASS audio).
2. Скопируйте `Spotify_Arizona_Rodina_Mobile.lua` в папку `moonloader`.
3. Зайдите в игру и введите `/spotify`.

При первом запуске скрипт создаёт папку `ST-Music` рядом с собой:

```
ST-Music/
  cache/             — кэш скачанных треков
  local/             — положите сюда свои .mp3 (плейлист "Local Files")
  user_playlists.txt — ваши личные плейлисты
  config.ini, likes.ini
```

## Как добавить свою музыку

Стриминги (Spotify и т.п.) отдают защищённый DRM-поток — прямой ссылки на
файл там нет. Поэтому используется хостинг прямых ссылок:

1. Скачайте трек на телефон (`.mp3`).
2. Загрузите его на [catbox.moe](https://catbox.moe) (или любой хостинг,
   дающий **прямую** ссылку на файл).
3. Скопируйте прямую ссылку.
4. В скрипте: вкладка **List** → кнопка **+** → введите название и ссылку →
   **Add**. Трек попадёт в ваш плейлист **My Music** и сохранится между
   сессиями.

Не хотите возиться со ссылками? Просто закиньте `.mp3` (или `.ogg`, `.wav`,
`.flac`) в папку `ST-Music/local` — они появятся в плейлисте **Local Files** и
играют без интернета.

## Что нового в 3.0

- Личные плейлисты: создание, добавление и удаление своих треков (работают
  кнопки **+ / OK / удалить**, раньше были заглушками).
- Воспроизведение **локальных файлов** из `ST-Music/local`.
- **Спектр-визуализатор** (кольцо вокруг диска, по FFT звука).
- **Таймер сна** (вкладка ℹ): авто-пауза через 15/30/60 минут.
- Вкладка ℹ с краткой инструкцией.
- Исправления: лайк-иконка теперь видна; текущий плейлист больше не
  сбрасывается на `default`; убран мёртвый код; устранён конфликт
  локальной/глобальной `curTrack`; меньше лишних запросов к Google Sheets.
